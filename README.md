## Installation ##
**Requirements:**
- Python >= 3.11

**How install it:**

`pip install dailytasks`

## Update process ##
1. Before update this package execute `dailytasks export` command to save all your tasks.

2. after update execute `dailytasks import` command to have all your tasks back.
